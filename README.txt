# Vaccine Prediction

This project predicts the likelihood of individuals receiving xyz and seasonal flu vaccines using machine learning techniques.

## Project Structure

- `main.py`: The main script for training the model and generating predictions.
- `data/`: Directory containing the training and test datasets.

## Requirements

- pandas
- scikit-learn

## How to Run

1. Ensure you have Python installed on your machine.
2. Install the required libraries:
    ```bash
    pip install pandas scikit-learn
    ```
3. Run the script:
    ```bash
    python main.py
    ```

## Submission

The `submission.csv` file will be generated in the project directory.
